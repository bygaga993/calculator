## src
calc.cpp - калькулятор, хранит в себе список операций
Operations.cpp - реализация операций, хранят внутри себ я информацию как их правильно преобразовывать во время парсинга и при получении результата
ExpressionParser.cpp - перевод строки в вектор токенов в порядке обратнйо польской нотации
main_func.cpp - функция с получением результата после всех преобразований входящей строки

## tests
tests_parser - тесты на корректность преобразования строки в вектор токенов
tests_opeartion - корректность работы бинарных операций
test_calculate - тестирование полноценных математических выражений
